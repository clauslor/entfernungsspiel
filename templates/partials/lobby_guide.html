📘 Spielanleitung

Alles Wichtige vor dem Start:

Spielregeln

  • Jede Runde: Distanz zwischen zwei Städten schätzen.
  • Standard: Wer am nächsten dran ist, bekommt 1 Punkt.
  • Der Host kann Sonderregeln aktivieren (z. B. erste Antwort beendet Runde).
  • Nach allen Runden gewinnt der Spieler mit den meisten Punkten.

Punktevergabe

  • Rundensieg: +1 Punkt für die beste Antwort.
  • Perfekttreffer-Bonus (nur Distanzfragen): +1 Punkt bei höchstens 20 km Fehler.
  • Streak-Bonus: Jeder 3. Rundensieg in Folge gibt +1 Zusatzpunkt.
  • Optionale Sonderregel: Falsche Antworten geben allen anderen je +1 Punkt (wenn vom Host aktiviert).
  • Sortierfragen: Die beste Reihenfolge gewinnt die Runde (+1 Punkt).

UI bedienen

  • Name setzen, dann Spiel erstellen oder über Code + PIN beitreten.
  • Host: Einstellungen prüfen/anpassen und ggf. sperren.
  • Alle Spieler klicken Bereit, dann startet die Runde.
  • Antwort im Feld eingeben und mit Antwort absenden bestätigen.
  • Shortcuts: N Name, J Join-Code, G direkt zum Eingabefeld.
